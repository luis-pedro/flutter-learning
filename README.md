# Flutter Studies

Repositório dedicado ao aprendizado de Flutter e Dart, com foco em prática e construção de pequenos projetos para consolidação de conceitos.

## 📌 Objetivo

Desenvolver habilidades em desenvolvimento mobile utilizando Flutter, aplicando conceitos como:

- Estrutura de widgets
- Layouts responsivos
- Navegação entre telas
- Gerenciamento de estado

## 📂 Organização

O repositório está dividido por tópicos de estudo, permitindo acompanhar a evolução prática ao longo do aprendizado.

## 🚀 Observação

Os projetos aqui têm caráter educacional e representam minha evolução contínua na tecnologia.
